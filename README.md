# 礼纪APP - 生命礼仪管家

## 📋 项目简介
专为生命礼仪师、殡葬主持人、礼仪团队打造的档期管理 + 客户管理 + 台词工具 + 团队协作工具。

## 🎯 核心功能
- ✅ 档期管理（日历视图、时间冲突检测）
- ✅ 客户管理（信息记录、历史服务）
- ✅ 工具箱（主持词、悼词模板、流程表）
- ✅ 文件上传（支持图片、文档）
- ✅ 数据统计（收入、场次、客户分析）

## 🚀 快速启动

### 1. 安装依赖
```bash
cd backend
npm install
```

### 2. 初始化数据库
```bash
cd ../database
node init.js
```

### 3. 启动服务器
```bash
cd ../backend
npm start
```

### 4. 访问应用
打开浏览器访问：http://localhost:3000

## 📁 项目结构
```
礼纪APP/
├── frontend/          # 前端文件
│   ├── index.html    # 主页面
│   ├── css/          # 样式文件
│   └── js/           # JavaScript文件
├── backend/          # 后端服务
│   ├── server.js     # Express服务器
│   └── package.json  # 依赖配置
├── database/         # 数据库
│   ├── init.js       # 初始化脚本
│   └── liji.db       # SQLite数据库
├── uploads/          # 上传文件目录
└── docs/             # 文档
```

## 🔧 技术栈
- **前端**：HTML5 + CSS3 + JavaScript (原生)
- **后端**：Node.js + Express
- **数据库**：SQLite3
- **文件上传**：Multer

## 📱 移动端访问
部署后，手机浏览器直接输入服务器地址即可访问。

## 🔒 安全说明
- 数据库文件请妥善保管
- 生产环境建议添加用户认证
- 敏感数据请勿明文存储
