### 📋 下一步操作指南

由于当前环境没有 AI 图片生成的 API Key，需要用户完成以下步骤：

#### 方案 1: Together AI (推荐)
1. 访问 https://api.together.ai/
2. 注册账号（支持邮箱或 Google 账号）
3. 进入 Settings → API Keys 创建新 Key
4. 运行:
   ```bash
   export TOGETHER_API_KEY='your-api-key'
   cd ~/Desktop/礼纪APP
   python3 generate_avatars.py
   ```

#### 方案 2: Replicate
1. 访问 https://replicate.com/
2. 注册账号（支持邮箱或 GitHub 账号）
3. 进入 Account Settings → API Tokens
4. 运行:
   ```bash
   export REPLICATE_API_TOKEN='your-token'
   cd ~/Desktop/礼纪APP
   python3 generate_avatars_replicate.py
   ```

#### 方案 3: Silicon Flow (国内服务)
1. 访问 https://siliconflow.cn/
2. 注册账号（支持手机号或邮箱）
3. 进入控制台 → API Keys
4. 运行:
   ```bash
   export SILICONFLOW_API_KEY='your-key'
   cd ~/Desktop/礼纪APP
   python3 generate_avatars_siliconflow.py
   ```
