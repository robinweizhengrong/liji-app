# 四川省殡仪馆数据收集指南

## 官方数据来源

### 1. 四川省民政厅官网
**网址：** http://mzt.sc.gov.cn/

**查找路径：**
1. 进入官网首页
2. 点击 "政务服务" 或 "便民查询"
3. 查找 "殡葬服务" 或 "殡仪馆名录"

### 2. 各市民政局官网
- 成都市民政局：http://cdmzj.chengdu.gov.cn/
- 绵阳市民政局：http://mzj.my.gov.cn/
- 达州市民政局：http://mzj.dazhou.gov.cn/
- 南充市民政局：http://mzj.nanchong.gov.cn/
- 泸州市民政局：http://mzj.luzhou.gov.cn/

### 3. 中国政府网查询
**网址：** https://www.gov.cn/
**查询方式：** 搜索 "四川省 殡仪馆"

## 收集方法

### 方法一：官网查询（推荐）
1. 访问各市民政局官网
2. 查找 "殡葬服务" 栏目
3. 下载或复制殡仪馆名录

### 方法二：电话咨询
1. 拨打各市民政局电话
2. 询问殡仪馆地址和联系方式

### 方法三：实地走访
1. 对已知的殡仪馆进行实地走访
2. 收集详细信息
