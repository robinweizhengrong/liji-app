# Kronos选股分析 - 预测回顾系统说明

## 📋 预测-回顾流程

### 第一步：每日收盘后（16:00-17:00）
**任务：生成预测**
- 分析当日行情
- 筛选次日关注股票
- 预测方向（涨/跌/平）
- 预测幅度（%）
- 记录置信度

**输出文件：** `predictions_YYYY-MM-DD.csv`

| 字段 | 说明 |
|------|------|
| 日期 | 预测日期 |
| 股票代码 | 股票代码 |
| 股票名称 | 股票名称 |
| 预测类型 | 短线(1-3天)/中线(1-4周)/长线(1-6月) |
| 预测方向 | 涨/跌/平 |
| 预测幅度 | 预计涨跌幅度（如：2-5%） |
| 预测理由 | 技术面/基本面/政策面等 |
| 置信度 | 70%/80%/90%等 |
| 信息来源 | K线/财报/新闻等 |

---

### 第二步：次日收盘后（16:00-17:00）
**任务：回顾验证**
- 获取实际收盘价
- 计算实际涨跌
- 对比预测 vs 实际
- 统计准确率

**输出文件：** `review_YYYY-MM-DD.csv`

| 字段 | 说明 |
|------|------|
| 预测日期 | 哪天做的预测 |
| 回顾日期 | 哪天做的回顾 |
| 股票代码 | 股票代码 |
| 预测方向 | 原预测方向 |
| 实际方向 | 实际涨跌方向 |
| 预测幅度 | 原预测幅度 |
| 实际幅度 | 实际涨跌幅 |
| 差距 | 预测与实际差值 |
| 是否正确 | ✅/❌ |
| 准确率 | 累计准确率 |
| 备注 | 特殊情况说明 |

---

### 第三步：每周/每月统计
**任务：生成报告**
- 统计周准确率
- 统计月准确率
- 分析预测方法效果
- 优化预测模型

**输出文件：** `accuracy_summary.csv`

| 统计周期 | 预测次数 | 正确次数 | 准确率 | 平均误差 |
|---------|---------|---------|--------|---------|
| 2026-W09 | 15 | 12 | 80% | 1.2% |
| 2026-02 | 45 | 36 | 80% | 1.5% |

---

## 📊 准确率计算方法

### 方向准确率
```
方向准确率 = 方向预测正确次数 / 总预测次数 × 100%
```

### 幅度准确率（更严格）
```
幅度误差 = |预测幅度 - 实际幅度|
幅度准确率 = (1 - 幅度误差/实际幅度) × 100%
```

### 综合评分
```
综合评分 = 方向准确率 × 0.6 + 幅度准确率 × 0.4
```

## 🎯 预测质量评估标准

| 等级 | 准确率 | 评价 |
|------|--------|------|
| 🏆 优秀 | ≥80% | 预测能力很强 |
| ✅ 良好 | 60-79% | 预测能力良好 |
| ⚠️ 一般 | 40-59% | 需要改进方法 |
| ❌ 较差 | <40% | 预测方法有问题 |
