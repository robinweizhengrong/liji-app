## 🚀 快速开始

### 方式1：一键启动（推荐）
```bash
cd ~/Desktop/礼纪APP
./start.sh
```

### 方式2：手动启动
```bash
# 1. 安装依赖
cd ~/Desktop/礼纪APP/backend
npm install

# 2. 初始化数据库
cd ../database
node init.js

# 3. 启动服务器
cd ../backend
npm start
```

### 访问地址
- 电脑：http://localhost:3000
- 手机：使用电脑IP地址（如 http://192.168.1.100:3000）

## 📝 使用说明

### 本地开发
1. 修改 `frontend/index.html` 调整界面
2. 修改 `frontend/js/api.js` 调整前端逻辑
3. 修改 `backend/server.js` 添加后端接口
4. 重启服务器生效

### 数据管理
- 数据库文件：`database/liji.db`
- 上传文件：`uploads/`
- 可以使用SQLite工具查看/编辑数据库

## 🎨 自定义修改

### 修改主题色
编辑 `frontend/index.html` 中的CSS变量：
```css
:root {
    --primary-color: #5a6c7d;  /* 主色调 */
    --bg-color: #f5f5f5;        /* 背景色 */
    ...
}
```

### 添加新功能
1. 前端：在 `index.html` 中添加新页面或组件
2. 后端：在 `server.js` 中添加新API路由
3. 数据库：在 `init.js` 中添加新表

## 🔒 安全提示
